Input Item Card Information


Manual: Enter Item Card information using the form editor, then click "Save".


Import: Click "Import Item Cards JSON" to bulk import Item Cards.



Preview and Print


Adjust Item Card sizes in the Table view. Check the Item Cards in the PDF view, then click "Export Item Cards PDF".


Open the exported PDF file in a PDF viewer, then Print on A4 paper.


Set print margins according to the printer for optimal print result.
