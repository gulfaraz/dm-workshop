Input Combat Token Information


Manual: Enter Combat Token information using the form editor, then click "Add Token".


Import: Click "Import Combat Tokens JSON" to bulk import Combat Tokens.



Preview and Print


Check the Combat Tokens in the Preview, then click "Export Combat Tokens HTML".


Open the exported HTML file in an internet browser, then Print on A4 paper.


Enable background graphics in the print settings and set print margins according to the printer for optimal print result.
